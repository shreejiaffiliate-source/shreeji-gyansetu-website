# Shreeji GyanSetu - Learning Management System

A cross-platform LMS featuring a Django backend and Flutter mobile application. Supports student queries, teacher responses, and real-time notification badges.

## 🚀 Getting Started

### 1. Prerequisites

- Python 3.10+
- Flutter SDK (for mobile app)
- Gmail account (for SMTP email notifications)

### 2. Backend Setup (Django)

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd shreeji-gyansetu
   ```
